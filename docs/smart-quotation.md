# 智能報價單功能

## 功能概述

讓業務透過自然語言描述，快速生成報價單草稿。系統自動比對客戶、產品，並提供歷史價格參考。

## 使用方式

### 從客戶頁面開啟（推薦）
在客戶詳情頁點擊「智能報價」按鈕，系統會自動帶入客戶資訊：

```
業務輸入：「報 2 台 CCR2004-16G-2S+，含一年 MA」

系統輸出：
┌─────────────────────────────────────────────────────────────┐
│ 客戶：國立台灣大學計資中心網路組 ✓ 100%                      │
│ 專案：MikroTik 設備採購                                       │
├─────────────────────────────────────────────────────────────┤
│ 產品                     │ 數量 │ 單價      │ 小計          │
│ CCR2004-16G-2S+ ✓ 100%   │ 2    │ $26,250   │ $52,500       │
│ 智慧網管 MA              │ 1    │ $31,104   │ $31,104       │
├─────────────────────────────────────────────────────────────┤
│                                        預估總額：$83,604    │
└─────────────────────────────────────────────────────────────┘
```

### 從客戶列表開啟
需要在描述中說明客戶名稱：

```
業務輸入：「幫台大報 2 台 MikroTik CCR2004，含一年 MA」
```

## API 端點

### POST /api/quotations/parse

解析自然語言報價描述。

**Request:**
```json
{
  "input": "幫台大報 2 台 MikroTik CCR2004，含一年 MA"
}
```

**Response:**
```json
{
  "customer": {
    "input": "台大",
    "matched": { "id": 123, "name": "國立台灣大學計資中心", "confidence": 0.85 },
    "suggestions": [...]
  },
  "project": "MikroTik 設備採購",
  "items": [
    {
      "input": "MikroTik CCR2004",
      "matched": { "id": "odoo-147", "name": "CCR2004-16G-2S+", "category": "MikroTik", "confidence": 0.95 },
      "quantity": 2,
      "priceUnit": 28000,
      "priceRange": { "min": 25000, "max": 30000, "avg": 27500 }
    }
  ],
  "totalAmount": 87104
}
```

### GET /api/quotations/parse?q=&type=

產品/客戶自動完成建議。

## 報價單分析發現

### 問題

1. **產品名稱無編碼** (444 次) - 需標準化
2. **規格過短/缺失** (222 次) - 需模板化
3. **價格差異大** (7 個產品) - 需價格管理

### 改進建議

1. 產品標準化 - 建立產品主檔 + 別名對照
2. 規格模板化 - 維護類/硬體類/軟體類
3. 價格管理 - 歷史價格 + 異常提醒
4. 客戶自動填入 - Odoo 連動
5. 專案模板 - IPPBX/智慧網管/SBC
